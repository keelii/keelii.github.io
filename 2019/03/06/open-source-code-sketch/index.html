代码画板 Code Sketch

最初写代码的地方…

功能

  • 内置 Sass/Babel 支持
  • HTML/CSS emmet 插件支持
  • 方便的导入三方库 (bootcdn API 支持).
  • 深色主题支持
  • 可以打印任意数据格式的控制台
  • 同时支持 macOS App 与 Web 端应用

快捷键

  • 切换展示顶部页签: Command+e
  • 代码字体大小: Command++/-
  • 保存刷新: Command+s
  • 导出到单个 HTML 文件: Command+shift+s
  • 导入文件（仅支持展出文件）: Command+o
  • 命令提示容器: Command+p
